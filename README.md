# Spotify Homepage Clone

A responsive Spotify Homepage Clone built using **HTML** and **CSS**. This project recreates the look and feel of Spotify's landing page, focusing on layout design, styling, and responsiveness.

## 🚀 Features

- Modern Spotify-inspired UI
- Responsive design for different screen sizes
- Navigation bar with menu options
- Hero section with attractive layout
- Music cards/playlists section
- Footer section
- Clean and organized code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
